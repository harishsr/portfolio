# GETTING STARTED WITH THE COMMAND LINE

Getting started on the command line can be scary.  Just knowing a few
commands, however, will make your experience MUCH easier and give you
confidence fast.

## CHEAT SHEET FOR BEGINNERS

```bash
man COMMAND   
# This command will give you the manual for a command
# Enter 'q' to quit

man sudo         # Learn about the sudo command
sudo COMMAND
# All UNIX commands are run under the current user's privileges and different
# users have permission levels.  The sudo command will run the command you
# specify under the root user (a 'superuser').  This is useful if you want to
# run a command and your user doesn't have access.  Use sparingly.

man pwd         # Learn about the pwd command
pwd             # Find current directory

man ls          # Learn about the ls command
ls              # List the contents of the current directory
ls -a           # List the contents of the current directory (even hidden files)
ls -al          # List all files AND list them vertically

man cd          # Learn about the cd command
cd DIRECTORY    # Change into a dir named DIRECTORY
cd path/to/dir  # Change into a deeper dir
cd ..           # Go up one directory
cd ../..        # Go up two directories
cd /            # Go to the root directory
cd ~            # Go to your user's home directory

man touch       # Learn about the touch command
touch FILENAME  # Create a file named FILENAME

man open        # Learn about the open command
open .          # Open the current dir in Finder

man mkdir       # Learn about the mkdir command
mkdir DIR       # Make a directory named DIR

man mv          # Learn about the mv command
mv FILE DIR/    # Move a file from the current directory to the DIR/ directory
mv FILE2 FILE2  # Rename FILE1 to FILE2

man cp          # Learn about the cp command
cp FILE DIR/    # Make a copy of FILE in DIR/

man rm          # Learn about the rm command
rm FILE         # Removes a file permanently
rm DIR/*.rb     # Removes all files ending in .rb from the DIR/ directory

# The following command is permanent so it requires thought before using it each time
rm -rf DIR      # Removes the DIR/ directory and all its contents permanently

man rmdir       # Learn about the rmdir command
rmdir DIR       # Removes the empty directory DIR permanently
```
